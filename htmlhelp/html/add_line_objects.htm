Add line objects
 

Press Add line component outline button
Press L
Press Add line placement outline button
Press l
Add line -> select layer

A line object will be added. When the spacebar is pressed, a dialogbox will popup, and the line parameters can be edited by hand. As many as 16 points (15 lines) can be edited. In addition, one point can be edited for the starting point of the line. When the first character typed is a @ the coordinates will be relative against the Relative (grid)position. The coordinates typed in will be used with the current units (dimension).
