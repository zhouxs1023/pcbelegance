Change colors
 

Sub menu View menu item Change colors

The color settings, brush and brush second color can be modified in the next dialogbox. The color settings will be copied into the pcb.ini initialisation file. This file is stored into the pcb subdirectory of the project.

To use those pcb colors for new designs, copy this pcb.ini file to main directory. Whenever a new design is created this pcb.ini file in the main directory will be copied to the pcb subdirectory of the new design.
