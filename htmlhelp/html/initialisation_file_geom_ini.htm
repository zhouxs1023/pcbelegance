Initialisation file geom.ini
 

The initialisation file geom.ini is used to save some designs parameters. The file will be in the same directory as the geometries.

The following paramaters are used:

[Settings]

WindowWidth The width of the windows
WindowHeight The height of the windows
WindowStartX Origin X of the windows

(0,0 = left top)
WindowStartY Origin Y of the windows
Units (0 = mils,1 = mm)
GridSize The gridsize (10nm units)
SnapMode 0 = FALSE,1 = TRUE
DrawGrid 0 = FALSE,1 = TRUE
